Mmail

A collection of emails

getCount()

Return a number of messages at the Mmail object.

		
			print m.getCount()
			>> 3

getSender()

Return a list of senders at the Mmail object.

		
			print m.getSender()
		>> ["Free food <vulture@mit.edu>", "Amy Zhang <axz@mit.edu>", "David Karger <karger@mit.edu>"]

getRecipient()

Return a list of recipients at the Mmail object.

		
			print m.getRecipient()
		>> ["Soya Park <soya@gmail.com>", "Amy Zhang <axz@mit.edu>", "David Karger <karger@mit.edu>"]

Not supported yet - getAttachment()

Return emails which have attachments.

Not supported yet - getContent()

Return a content of emails at the Mmail object.

Not supported yet - getFlags()

Return a content of emails at the Mmail object.

getSubject()

Return a list of subject of the messages at the Mmail object

		
			print m.getSubject()
			>> ["Free food at G5!", "Please read my draft", "FINAL HOURS: 20% OFF Sitewide!"]

queue

If you enable this feature, your email clients don't give you notification until your defined queue is full. Once queue is full, it gives one notification, empty the queue and then repeat the process.

Parameter
is_queue_full function is_queue_full will be passed a Mmail object which captures incoming emails after queue() has been called. is_queue_full should return boolean value, True when the queue should be flushed and False otherwise. It must not be null.
folders list of string Which folder to digest. If an email arrives at other folder, they wouldn't added to the queue.
digest_message function What to include at a digest email when the queue is full. 

		
			def is_queue_full(inComingMmail):
				if "Deadline" in inComingMmail.getSubject():
					return True
				return False
